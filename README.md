# nlp-sentiment-analysis
Basic sentiment analysis of Shakespeare text using Python and simple NLP techniques.

## Approach
- Selected sample text from Shakespeare’s works  
- Converted text to lowercase  
- Removed punctuation  
- Split sentences into words  
- Used predefined positive and negative word lists  
- Compared words and classified sentiment based on counts  

## Difficulties Faced
- Understanding how to clean and process text  
- Deciding sentiment without using any machine learning model  
- Handling sentences with mixed emotions  

## Resolutions
- Broke the problem into smaller steps  
- Used simple Python functions for preprocessing  
- Applied a rule-based approach using word matching  

## Results
- Each sentence was classified as Positive, Negative, or Neutral  
- The output was based on the number of positive and negative words  

## Learnings
- Learned basic NLP concepts like text cleaning and tokenization  
- Understood how rule-based sentiment analysis works  
- Gained insight into limitations of simple models  

## How to Run
1. Make sure Python is installed  
2. Run the file: sentiment_analysis.py

## Data Source
Text taken from works of William Shakespeare such as:
- Romeo and Juliet  
- Macbeth
